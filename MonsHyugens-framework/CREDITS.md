# 🛰️ Sputnik Framework - Créditos

## 🙏 Créditos ao Lua

Este framework é construído sobre a linguagem **Lua** - uma poderosa linguagem de script leve e rápida.

### 📜 Informações sobre Lua

**Lua** foi criada e é mantida por **PUC-Rio** (Pontifícia Universidade Católica do Rio de Janeiro).

- **Site Oficial:** [https://www.lua.org](https://www.lua.org)
- **Licença:** MIT License
- **Licença Completa:** [https://www.lua.org/license.html](https://www.lua.org/license.html)

### 🌟 Por que usamos Lua?

Lua é escolhida para o Sputnik Framework por suas características excepcionais:

- **Leve e Rápida:** Performance excepcional com footprint mínimo
- **Portabilidade:** Roda em praticamente qualquer plataforma
- **Simplicidade:** Sintaxe limpa e fácil de aprender
- **Extensibilidade:** Fácil de estender com C e Assembly
- **Embeddable:** Perfeita para integração em sistemas maiores
- **Comunidade Ativa:** Comunidade global robusta e suporte excelente

### 🎯 Como dar crédito ao Lua

Conforme a licença MIT do Lua, ao usar o Sputnik Framework (que utiliza Lua), você deve:

1. **Incluir o aviso de copyright** em sua distribuição
2. **Manter a licença MIT** intacta
3. **Dar crédito apropriado** aos criadores do Lua

### 📋 Modelo de Crédito

Ao usar o Sputnik Framework em seus projetos, recomendamos incluir o seguinte crédito:

